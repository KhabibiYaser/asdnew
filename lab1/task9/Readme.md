 ## Задание 9
 
 Сложить два n-битовых двоичных числа, хранящихся в массиве, и вывести результат как (n+1)-битовый массив.
 
 ## Input / Output
 
 | Input               | Output |
 | ------------------- | ------ |
 | `1011`              | `11000`|
 | `1101`              |        |
 
 ## Ограничения
 
$$
( 1 \leq n \leq 1000 )
$$
