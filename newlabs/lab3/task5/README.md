# Задача 5: `Индекс Хирша`
 
## Задание

Рассчитан индекс Хирша для массива целых чисел, представляющих количество цитирований научных статей.

## Input / Output

| Input     | Output |
|-----------|-----|
| `1, 3, 1` | `1` |

## Ограничения

$$
( 1 \leq n \leq 100,000 )
$$
