 ## Задание 1
 
 Реализовать алгоритм сортировки вставкой для сортировки массива в порядке возрастания.
 
 ## Input / Output
 
 | Input           | Output          |
 | --------------- | --------------- |
 | `6`             | `26 31 41 41 58 59` |
 | `31 41 59 26 41 58` |                |
 
 ## Ограничения
 
 - \( 1 \leq n \leq 1000 \)
 - \( |a_i| \leq 10^9 \)
