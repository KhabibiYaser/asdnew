 ## Задание 6
 
 Реализовать алгоритм пузырьковой сортировки для сортировки массива в порядке возрастания.
 
 ## Input / Output
 
 | Input           | Output          |
 | --------------- | --------------- |

 | `31 41 59 26 41 58` |                |
 
 ## Ограничения
 $$
( 1 \leq n \leq 1000 )
$$
$$
( |a_i| \leq 10^9 )
$$
