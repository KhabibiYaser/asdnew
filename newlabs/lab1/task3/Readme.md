 ## Задание 3
 
 Реализовать алгоритм сортировки вставкой для сортировки массива в порядке убывания.
 
 ## Input / Output
 
 | Input           | Output          |
 | --------------- | --------------- |
 | `6`             | `59 58 41 41 31 26` |
 | `31 41 59 26 41 58` |                |
 
 ## Ограничения
$$ 
( 1 \leq n \leq 1000 )
$$
$$
  ( |a_i| \leq 10^9 )
$$
