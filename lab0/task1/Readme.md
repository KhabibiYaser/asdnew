## Задание 1

Даны два числа \( a \) и \( b \), требуется:
1. Найти их сумму.
2. Вывести результат на экран.
3. Организовать ввод данных через файлы.

## Input / Output

| Input         | Output |
| ------------- | ------ |
| `2 3`         | `5`    |
| `-10 15`      | `5`    |

## Ограничения

$$
-10^9 \leq a, b \leq 10^9
$$
- Время выполнения: 1 сек.
- Память: 256 Мб.
