 # Задача 6: `Сортировка произведений`
 
 ## Задание
 
 Выполнена сортировка произведений чисел из двух массивов. Затем рассчитана сумма каждого десятого элемента отсортированной последовательности.
 
 ## Input / Output
 
 | Input           | Output                                   |
 | --------------- | ---------------------------------------- |
 | `5 5`           | `Сумма: 12345`                          |
 | `1 2 3 4 5`     |                                          |
 
 ## Ограничения

 
$$
( 1 \leq n, m \leq 100,000 )
$$
